En restaurante la cabaña, también tenemos servicios especiales para cualquier tipo de evento, como los siguientes:

*Reservación completa del restaurante: Es el indicado para cualquier tipo de evento ejecutivo o algún tipo de fiesta, este servicio incluye:

    • Mesas.
    • Sillas.
    • Cristalería
    • Fundas para sillas.
    • Manteles.
    • Centros de mesa.
    • Diferentes menús para elegir.
    • Música al gusto

    *Servicios de Banquete: Es parecido a la reservacion completa del restaurante, con la diferencia de que en este nosotros llevamos todo lo que sea necesario para un buen banquete al lugar donde el cliente lo necesite, este incluye:

    • Mesas.
    • Sillas.
    • Cristalería
    • Fundas para sillas.
    • Manteles.
    • Centros de mesa.
    • Diferentes menús para elegir.
    • Música al gusto

    *Reparto a domicilio dentro de la cabecera municipal y aldeas cercanas.
